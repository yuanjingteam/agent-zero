什么是 AI 智能体？

AI 智能体不仅仅是文本生成 — 它是 LLM 与工具、记忆和推理模式的结合。

智能体公式 AI 智能体 = LLM + 系统提示 + 工具 + 记忆 + 推理模式

大脑      身份标识      双手      状态      策略
🧠

LLM（大脑）

处理信息、生成响应、做出决策的核心引擎。本地运行，数据不离开你的机器。

🎯

系统提示（身份）

定义智能体的角色、行为和约束。让通用模型变成专业助手。

🔧

工具（双手）

函数调用让 LLM 能够执行操作 — 查询时间、计算、搜索、操作文件。

💾

记忆（状态）

跨会话持久化信息，让智能体记住用户偏好和历史交互。

快速开始

三步即可开始学习。

# 1. 克隆仓库
git clone https://github.com/yuanjingteam/agent-zero.git
cd agent-zero

# 2. 安装依赖
npm install

# 3. 下载模型（选择一个即可）
npx node-llama-cpp pull qwen3-1.7b

# 4. 运行第一个示例
node examples/01_intro/intro.js
